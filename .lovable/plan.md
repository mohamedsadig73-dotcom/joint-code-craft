## المشكلة الحالية

في صفحة **التعبئة (Packing)**، اختيار الأصناف لطباعة ورقة التعبئة يتم **قطعة قطعة** بـ Checkbox واحدة لكل صف، وهذا بطيء جداً عندما يكون عندك مئات الأصناف (مثل: 689 جاهز للتعبئة).

سير العمل المطلوب:
1. اختيار الأصناف بسرعة
2. طباعة ورقة تعبئة (Packing Sheet) فيها عمود **رقم الصندوق فارغ** ليُملأ يدوياً
3. بعدها يُدخَل رقم الصندوق في النظام دفعة واحدة

## الحل المقترح — تعبئة احترافية بثلاث طبقات

### 1. اختيار سريع متعدد الأبعاد (Multi-dimensional selection)

بدلاً من checkbox واحدة لكل صنف، نضيف شريط أدوات اختيار ذكي فوق القائمة:

```text
┌─────────────────────────────────────────────────────────────┐
│  [☐ اختيار الكل المرئي]   تحديد سريع: ▾                    │
│                                                             │
│  حسب الوجهة:    [المغرب 245]  [أوزبكستان 412]  [غير محدد]  │
│  حسب المورد:    [Combobox: شركة...] + زر "اختر كل أصناف"   │
│  حسب رقم الفاتورة: [Combobox: INV-001...] + "اختر الفاتورة"│
│  حسب التصنيف:   [boxed 600]  [loose 89]                    │
│                                                             │
│  [مسح التحديد]                            المحدد: 245 صنف  │
└─────────────────────────────────────────────────────────────┘
```

**أزرار التحديد السريع** (chips قابلة للضغط):
- اضغط chip "المغرب" → يحدد كل أصناف المغرب المرئية فوراً
- اضغط chip "اختر فاتورة INV-001" → يحدد كل الأصناف المرتبطة بهذه الفاتورة
- اضغط chip "اختر مورد X" → يحدد كل أصناف هذا المورد
- يمكن دمج الفلاتر (المغرب + مورد X)

### 2. وضع التحديد بالسحب (Drag-to-select / Shift+Click)

- **Shift+Click** على صنف لتحديد كل ما بينه وبين آخر صنف محدد (مثل Excel/Gmail)
- **اختصار لوحة المفاتيح:** `Ctrl+A` لاختيار الكل المرئي، `Esc` لمسح التحديد
- **ضغطة طويلة على الجوال** + سحب لتحديد عدة عناصر دفعة واحدة

### 3. ورقة التعبئة القابلة للطباعة (Packing Worksheet)

زر جديد بارز: **"🖨️ طباعة ورقة التعبئة"** يظهر عندما يكون عدد المحددات > 0.

ينتج صفحة A4 فيها:

```text
┌──────────────────────────────────────────────────────────────┐
│  ورقة تعبئة — التاريخ: 10/05/2026     عدد الأصناف: 245      │
│  المسؤول: ___________________  التوقيع: ___________          │
├────┬─────────────┬──────────────────────┬─────┬─────┬────────┤
│ #  │ رقم الصنف   │ الوصف                │ الكم │ الوحدة│ صندوق │
├────┼─────────────┼──────────────────────┼─────┼─────┼────────┤
│ 1  │ TCP02       │ 315/70 R 17 ...      │ 40  │ PCS │ B-___ │
│ 2  │ MAK/A/D-066 │ Drill Bit HSS...     │ 1   │ SET │ B-___ │
│ ...│             │                      │     │     │ B-___ │
└────┴─────────────┴──────────────────────┴─────┴─────┴────────┘
```

- مجموعة حسب الوجهة (المغرب أولاً، ثم أوزبكستان...)
- خانة **"رقم الصندوق"** تكون خطوط فارغة كبيرة سهلة الكتابة باليد
- باركود/QR صغير في الأعلى يحوي معرّفات الأصناف (للمرحلة التالية: المسح وإدخال الأرقام تلقائياً)
- زر تنزيل PDF + زر طباعة مباشرة

### 4. إدخال أرقام الصناديق دفعة واحدة بعد التعبئة الفعلية (Bulk Box Assignment)

بعد ما يعبّي العامل ويكتب أرقام الصناديق على الورقة، يفتح حواراً جديداً **"إدخال أرقام الصناديق"**:

```text
┌─────────────────────────────────────────────────┐
│  إدخال أرقام الصناديق دفعة واحدة                │
│  ─────────────────────────────────────────       │
│  وضع الإدخال:   ◉ صف-بصف   ○ تجميعي             │
│                                                  │
│  TCP02       — 315/70 R 17...    [B-01]        │
│  MAK/A/D-066 — Drill Bit HSS...  [B-01]        │
│  MAK/A-D-067 — Drill Bit HSS...  [B-02]        │
│  ...                                            │
│                                                  │
│  💡 نصيحة: اكتب الرقم واضغط Enter للانتقال     │
│  💡 أو اختر عدة صفوف ثم "تعيين كلها لـ B-__"    │
│                                                  │
│         [إلغاء]    [حفظ الكل وتعليم كمعبأ]      │
└─────────────────────────────────────────────────┘
```

- وضع **"تجميعي"**: حدد عدة صفوف → اكتب رقم صندوق واحد → يُطبَّق على كلها (مهم لما يكون عدة أصناف في نفس الصندوق)
- وضع **"صف-بصف"**: Tab/Enter ينتقل تلقائياً للصف التالي (سرعة كاتب)
- التحقق من صيغة `B-XX` تلقائياً (يقبل "1" → يحوّلها "B-01")
- زر "حفظ الكل" يقوم بـ `bulkUpdateFields` واحدة لتغيير جميع الصناديق + تغيير الحالة إلى `packed`

## التفاصيل التقنية

**ملفات جديدة:**
- `src/components/boxes/packing/PackingQuickSelectBar.tsx` — شريط الاختيار السريع بالـ chips
- `src/components/boxes/packing/PackingWorksheetPrint.ts` — مولد HTML لورقة التعبئة A4 (نفس نمط `buildSupplierInvoiceHTML.ts`)
- `src/components/boxes/packing/BulkBoxAssignDialog.tsx` — حوار إدخال أرقام الصناديق دفعة واحدة
- `src/hooks/usePackingSelection.ts` — hook لإدارة التحديد + Shift+Click + اختصارات

**ملفات معدّلة:**
- `src/components/boxes/PackingTab.tsx` — استبدال شريط الفلاتر القديم بالنظام الجديد، إضافة الزرين الجديدين
- `src/locales/ar.ts` و `src/locales/en.ts` — إضافة المفاتيح الجديدة (~15 مفتاح)

**بدون تغييرات في قاعدة البيانات** — كل المنطق في الواجهة، نستفيد من `bulkUpdateFields` الموجود في `useBoxReceipts`.

**الطباعة:** نستخدم نفس استراتيجية `iframe + Blob` للويب و `webContents.printToPDF` للـ Electron (متوافق مع معيار النظام).

## سير العمل بعد التطبيق

1. تفتح التبويب → ترى 689 صنف
2. تضغط chip **"المغرب"** → يحدد 245 صنف فوراً
3. تضغط **"🖨️ طباعة ورقة التعبئة"** → يطبع A4 منظمة بخانة صندوق فارغة
4. العامل يعبّي ويكتب أرقام الصناديق يدوياً على الورقة
5. ترجع للنظام، تضغط **"إدخال أرقام الصناديق"** → تكتب الأرقام بسرعة (Enter للتالي) أو تجمع عدة أصناف لنفس الصندوق
6. تضغط **"حفظ"** → الكل يصبح `packed` بأرقام الصناديق الصحيحة دفعة واحدة

النتيجة: من ~5 دقائق لاختيار 200 صنف → أقل من 10 ثوانٍ.
