الخطة المقترحة لمعالجة المشكلة:

1. تمكين تصحيح رقم القطعة في سجل الاستلام
- إزالة القفل الحالي الذي يمنع تغيير رقم القطعة عند تعديل سجل موجود.
- عند اختيار رقم قطعة صحيح من قاموس الأصناف، سيتم تحديث السجل بالقيم الصحيحة:
  - رقم القطعة
  - الوصف
  - الوحدة
  - صورة الصنف إن وجدت
  - ربط السجل بالصنف الصحيح في قاموس الأصناف
- إبقاء التعديل محكوماً بنفس الصلاحيات الحالية: صاحب السجل أو المدير/الأدمن فقط.

2. إصلاح الربط مع قاموس الأصناف عند التعديل
- إضافة `item_id` إلى بيانات سجل الاستلام في الواجهة حتى لا يبقى السجل مرتبطاً بالصنف القديم بعد تغيير رقم القطعة.
- تعديل دالة تحديث السجل لترسل `item_id` الصحيح مع `part_no`.
- إضافة حماية في قاعدة البيانات لضمان أن أي سجل يتم إدخاله أو تحديثه يتم ربطه تلقائياً بالصنف الصحيح إذا لم يُرسل `item_id` من الواجهة.

3. إضافة استرجاع الفاتورة كاملة للتعديل
- حالياً الفاتورة الكاملة تُحفظ كأسطر منفصلة، ورقم الفاتورة موجود داخل الملاحظات فقط، لذلك لا يمكن فتحها كفاتورة واحدة للتعديل.
- سأضيف واجهة لاختيار/فتح فاتورة كاملة من السجلات الموجودة حسب رقم الفاتورة.
- عند فتحها سيتم تحميل كل الأسطر المرتبطة بنفس رقم الفاتورة داخل نموذج الفاتورة الكاملة، بدل تعديل كل سطر منفرداً.
- يمكن تعديل رأس الفاتورة دفعة واحدة مثل:
  - المورد
  - التاريخ
  - رقم الفاتورة
  - الوجهة
  - نوع التعبئة
  - رقم الصندوق/المكان
  - الحالة
- ويمكن تعديل الأسطر مثل:
  - تغيير رقم القطعة
  - الوصف
  - الكمية
  - الوحدة
  - الملاحظات
  - حذف سطر
  - إضافة سطر جديد

4. تحسين طريقة حفظ رقم الفاتورة
- إضافة حقل قاعدة بيانات مخصص لرقم الفاتورة في `box_receipts` بدلاً من الاعتماد على نص داخل الملاحظات فقط.
- ترحيل البيانات القديمة تلقائياً: أي سجل يحتوي ملاحظة بالشكل الحالي `[رقم الفاتورة: ...]` سيتم استخراج الرقم منه وتعبئته في الحقل الجديد.
- تعديل البحث والجدول ليشمل رقم الفاتورة حتى تستطيع الوصول لها بسهولة.

5. واجهة الاستخدام
- إضافة زر/خيار واضح مثل: "تعديل فاتورة كاملة" بجانب زر "فاتورة كاملة".
- إضافة قائمة فواتير تعرض رقم الفاتورة، المورد، التاريخ، وعدد الأسطر.
- عند اختيار فاتورة، يفتح نفس نموذج الفاتورة لكن بوضع "تعديل" بدلاً من "إضافة".
- إضافة رسائل عربية واضحة لنجاح التعديل أو عند عدم العثور على فاتورة.
