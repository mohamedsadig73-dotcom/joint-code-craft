# خطة إعادة ربط GitHub بمشروع DTS-Store

## 🎯 الهدف
استعادة المزامنة الثنائية بين Lovable ومستودع `mohamedsadig73-dotcom/joint-code-craft` على GitHub.

## 🔍 التشخيص المؤكد
- **من جهة Lovable**: لا يوجد أي اتصال GitHub مسجّل في حسابك حالياً (تم التحقق عبر فحص الاتصالات).
- **من جهة GitHub**: حسابك نشط والمستودع موجود (423 commit في أبريل 2026).
- **السبب الحقيقي**: عند الضغط على "Uninstall" في GitHub سابقاً، تم حذف الربط من الجانبين، وإعادة التثبيت من GitHub وحدها **لا تُعيد إنشاء الربط** — يجب أن تبدأ العملية من داخل Lovable.

## 🛠️ خطوات الحل (بالترتيب الصارم)

### الخطوة 1: التنظيف من جهة GitHub أولاً
1. اذهب إلى: https://github.com/settings/installations
2. إذا رأيت تطبيق **Lovable** في القائمة → اضغط **Configure** → انزل لأسفل → **Uninstall**
3. تأكد أن GitHub لم يعد يحتوي على أي بقايا للتطبيق القديم.

### الخطوة 2: بدء الربط من داخل Lovable (الأهم)
1. في Lovable، افتح القائمة الجانبية → **Connectors** (المستوى الجذري)
2. ابحث عن **GitHub** → اضغط **Connect project**
3. ستفتح نافذة منبثقة لـ GitHub → سجّل الدخول بحساب `mohamedsadig73-dotcom`
4. اختر: **All repositories** (مؤقتاً، لتفادي أي خطأ في الاختيار)
5. اضغط **Install & Authorize**

### الخطوة 3: إنشاء/ربط المستودع
- بعد العودة إلى Lovable، إذا طُلب منك:
  - **"Create Repository"** → اضغطها لإنشاء مستودع جديد
  - **"Select existing repository"** → اختر `joint-code-craft`

### الخطوة 4: التحقق من نجاح الربط
- ستظهر رسالة خضراء "Connected" داخل Lovable
- جرّب أي تعديل بسيط في Lovable وتحقق من ظهوره في GitHub خلال دقيقة

## ⚠️ إذا فشلت الخطوات أعلاه

في هذه الحالة سأقوم بـ:
1. تشغيل أداة التشخيص الداخلية في Lovable (`standard_connectors--connect`) التي ستفتح لك نافذة الربط مباشرة من داخل المحادثة دون الحاجة للذهاب إلى Connectors يدوياً.
2. إذا استمر الفشل → أتواصل مع Lovable Support نيابة عنك لأن المشكلة ستكون على مستوى المنصة.

## 📌 ملاحظة مهمة
هذه الخطة **لا تتطلب أي تعديل على الكود** — فقط إعادة ضبط الربط. مشروعك ومستودعك آمنان تماماً، ولن تفقد أي ملف.

**هل توافق على المتابعة؟** بمجرد الموافقة، سأبدأ بتشغيل أداة `standard_connectors--connect` التي ستعرض لك نافذة الربط مباشرة وتُسهّل العملية.
